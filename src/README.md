## 目录说明

### assets 
放置图片等资源

### components
仅适用于本项目的组件
 * /pages 存储路由视图组件

### idxLib
我们自己开发的一些可复用组件和工具，俗称轮子
 * /components 可复用组件，详情看该文件下的readme.md
 * /router 可复用的路由表
 * /utils 工具库，详情看该文件下的readme.md
 * /views 可复用的视图

### router.js
本项目的路由

### store.js
vuex

### main.js 
vue页面的起点
